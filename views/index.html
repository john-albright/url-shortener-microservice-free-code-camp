URL Shortener Microservice

Project Description

This is project 3 of 5 for FreeCamp.org's APIs and Microservices certificate, the 5th of 6 full stack JavaScript certificates. This API creates short urls for valid websites by assigning a number to the website and allowing the website to be accessed from the subdirectory api/shorturl/ followed by the number of the website. For starters, the subdirectory 1 (/api/shorturl/1) will take you to the search engine DuckDuckGo.
